解决国内访问github速度太慢的问题

解决方法：

1.访问http://github.global.ssl.fastly.net.ipaddress.com/#ipinfo

可以获取域名以及ip，如以下内容

Fastly DNS Resource Records
Name Type Data TTL
github.global.ssl.fastly.net A 151.101.113.194 30

打开hosts文件(C:\Windows\System32\drivers\etc)中，添加：

52.74.223.119    github.com
151.101.113.194  github.global.ssl.fastly.net

打开速度贼快，好鸡冻
